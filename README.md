# React App with Redux and Authentication

This is a React application built with Redux for state management, including user authentication and product CRUD operations using the DummyJSON API.

## How to Run

1. Navigate to the project directory:
   ```
   cd myapp
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

The app will open in your browser at [http://localhost:3000](http://localhost:3000).
